This program uses Liberica JDK 11.0.10 provided by BellSoft 
since this specific version of JDK 11 includes JavaFX.

This program was coded in IntelliJ IDEA Community Edition 2020.3 
but I have adapted it to be command line runnable. 

Extract this ZIP folder then do the following:

  1) Ensure you have JDK 11.0.10 BellSoft Liberica installed on your system.
  2) Point your JAVA_HOME Environment variable to this specific JDK.
  3) Open Command Prompt and navigate to the extracted ZIP folder.
  4) Compile code; type: javac *.java
  5) Run the application; type: java Driver
